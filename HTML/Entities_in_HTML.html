HTML Entities

HTML entities are special codes used to represent characters that have a specific meaning in HTML or are not easily typed on a keyboard. These entities are used to ensure that the characters are displayed correctly in the browser without being interpreted as HTML code. They begin with an ampersand (&) and end with a semicolon (;).

Uses of HTML Entities:

    • Display Reserved Characters: Characters like <, >, &, and " have specific meanings in HTML (e.g., tag delimiters). Entities allow these characters to be displayed as text.
      Display Special Characters: Characters that are not available on a standard keyboard or in the ASCII set, like © (copyright), ™ (trademark), and others.
      Encoding Non-ASCII Characters: Ensures that characters from various languages are displayed correctly on all browsers.

    • Common HTML Entities:

      '<' - Less than (<)
      '>' - Greater than (>)
      '&' - Ampersand (&)
      ' " ' - Double quote (")
      ''' - Single quote (')
      '©' - Copyright (©)
      '®' - Registered trademark (®)
      ' ' - Non-breaking space ( )
